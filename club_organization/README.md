# NAU Student Organizations Web App

## Description
A modern, responsive front-end website for student organizations at North American University. It highlights active clubs, upcoming events, and provides forms for membership registration and contact.

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla)

## Features
- Hero section with description and call-to-action button  
- Scrollable student organization card slider  
- Upcoming events slider with registration form  
- Membership registration with validation  
- "Get in Touch" contact section  
- Fully responsive layout  
- Sidebar menu for mobile navigation  
- Footer with quick links and NAU branding
